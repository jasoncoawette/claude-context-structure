# Out of Office

## What this does

Helps the user prepare for time away: setting OOO dates, drafting auto-replies, arranging coverage, and ensuring a clean handoff.

## When to use

- The user is planning a trip or time off
- An auto-reply or OOO calendar block needs setting up
- Work or commitments need coverage during the absence

## Process

1. Confirm the OOO start and end dates and the timezone.
2. Block the calendar and draft an auto-reply with a return date.
3. Note who covers what and any urgent items to wrap before leaving.

## Common pitfalls

- Setting an auto-reply with the wrong return date.
- Leaving open commitments without arranging coverage.

## Scripts

- `Script.py` — generate an OOO auto-reply message from start/end dates.
