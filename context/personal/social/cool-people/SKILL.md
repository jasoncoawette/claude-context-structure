# Cool People

## What this does

A lighter-touch contact list of acquaintances and interesting people the user has met and wants to nurture into stronger connections.

## When to use

- The user meets someone new worth remembering
- The user wants to follow up with an acquaintance
- Looking for someone to invite to an event or activity

## Process

1. Capture how the user met the person and what stood out.
2. Note shared interests and any follow-up intent.
3. Suggest a low-pressure way to stay in touch when relevant.

## Common pitfalls

- Letting promising connections go cold after one meeting.
- Logging vague notes that don't jog memory later.

## Scripts

- `Script.py` — list acquaintances with no follow-up logged yet.
