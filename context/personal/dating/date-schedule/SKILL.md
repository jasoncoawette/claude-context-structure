# Date Schedule

## What this does

Keeps the user's dating calendar organized: upcoming dates with who/when/where, and a light history to track how things are going.

## When to use

- The user schedules, reschedules, or cancels a date
- The user asks what's coming up on the dating calendar
- Logging notes after a date

## Process

1. Capture the person, date, time, and location.
2. Check for calendar conflicts before confirming.
3. Set a reminder and log a brief note afterward.

## Common pitfalls

- Double-booking or scheduling too tight back to back.
- Mixing up details between different people.
